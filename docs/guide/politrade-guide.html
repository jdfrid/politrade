1. מה זה Politrade?

Politrade הוא לוח בקרה (Dashboard) לבוט שמתחבר ל-Polymarket דרך CLOB API. המערכת מאפשרת:

  • הימורי Crypto 5 דקות — Up/Down על BTC/ETH/SOL/XRP לפי מחיר Chainlink
  • Portfolio — צפייה בפוזיציות, Cash ו-PnL כמו באתר Polymarket
  • רכישה ומכירה — מסחר ידני בשווקים רגילים ובחלונות Crypto
  • בוט אוטומטי — הימור בחלון דקות 3–4 כשיש edge מינימלי

2. התחלה מהירה

בענן (Render)

כתובת: https://politrade-web.onrender.com

התחברות: שם משתמש admin + סיסמה מ-DASHBOARD_PASSWORD ב-Render.

חשוב: Render בארה"ב — Polymarket עלול לחסום מסחר (geoblock). צפייה ב-Portfolio עובדת; לקנייה/מכירה הרץ מקומית.

מקומית (Windows)

  1. פתח PowerShell בתיקיית הפרויקט
  2. הרץ: .\start-local.ps1
  3. פתח: http://localhost:8000
  4. הגדר ארנק ב-/wallet

4. שלב ① — חיבור ארנק (/wallet)

כתובת נכונה: מ-Polymarket → Settings → Profile → Address ("For API use only").
לא Relayer Signer ולא Relayer API Key.
  1. היכנס ל-polymarket.com/settings
  2. העתק Deposit Address (42 תווים, מתחיל ב-0x)
  3. הדבק ב-Funder — לחץ Copy, אל תקליד ידנית
  4. הזן Private Key של הארנק (EOA)
  5. בחר Signature Type 3
  6. לחץ שמור חיבור
  7. בצע עסקה אחת ידנית ב-Polymarket לפני הבוט

6. Portfolio — פוזיציות ומסחר

6.1 רכישה בשוק רגיל (Brazil, בחירות…)

  1. מצא את slug השוק — מ-Polymarket או מעמודת הפוזיציה
  2. הדבק בשדה "רכישה" → טען שוק
  3. בחר outcome (Yes/No) + סכום ב-$
  4. לחץ קנה

דוגמת slug: fifwc-bra-jpn-2026-06-29-bra

6.2 מכירת פוזיציה

  1. בטבלת Positions מצא את השורה
  2. לחץ מכור
  3. הזן כמות shares (Enter = הכל)
  4. אשר — המערכת שולחת market sell ל-CLOB

6.3 כפתור + (קנייה נוספת)

טוען אוטומטית את slug השוק לטופס הרכישה.

6.4 History

מציג עסקאות מ-Polymarket, הימורי Crypto, ושגיאות — לפי זמן.

7. כל הימורי Crypto 5m הפתוחים

טבלה עם BTC · ETH · SOL · XRP — עד 8 חלונות קדימה (~40 דקות).

עמודההסבר
נכסBTC / ETH / SOL / XRP
חלוןטווח זמן UTC + שם השוק
שלבמוקדם / חלון הימור / מאוחר
Up / Downמחיר ask + edge %
ניתן לקנות?כן — או סיבת חסימה
פעולותכפתורי Up / Down

מתי מופיע "כן" לקנייה?

  • CLOB מחובר + יתרה מספקת
  • שוק לא סגור + יש נזילות
  • לא הימרת כבר באותו חלון (חלון אחד = הימור אחד)

איך קונים מהטבלה?

  1. לחץ Up או Down בשורה הרצויה
  2. הזן סכום ב-$ (ברירת מחדל מההגדרות)
  3. העסקה נשלחת — הטבלה מתרעננת

8. חלונות BTC חיים והימור אוטומטי

שלבי חלון 5 דקות

שלבזמןבוט אוטומטי
מוקדם (early)דקות 0–2לא מהמר
חלון הימור (bet)דקות 3–4מהמר אם edge ≥ 15%
מאוחר (late)דקה 5לא מהמר

מושגים בכרטיס החלון

  • Price to Beat — מחיר Chainlink בתחילת החלון
  • Chainlink — מחיר נוכחי + אחוז שינוי
  • Up/Down ask — מחיר מניה; מניה זוכה = $1.00
  • edge — רווח פוטנציאלי: (1 − ask) / ask

כפתורי בקרה

כפתורפעולה
הפעל אוטומטיהבוט מהמר לבד בחלון bet
כבה אוטומטירק צפייה + הימור ידני
המר UP/DOWN מומלץהימור ידני לפי המלצת האסטרטגיה
נוסחת תשלום: השקעה ÷ מחיר ask = מניות. אם צדקת: מניות × $1.00.
